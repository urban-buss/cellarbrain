## 💪 Power-Prompts — Komplexe Anfragen

Mehrteilige oder situative Anfragen, die mehrere Skills kombinieren:

**Wochenplanung:**
```
Was soll ich diese Woche trinken? 3 Vorschläge für Wochentage
(casual, allein) und 1 für Samstag (Gäste, italienisch kochen)
```

**Dinner mit vollem Briefing:**
```
Dinner am Samstag für 6: Tatar | Steinbutt | Lammkarree | Époisses.
Budget 200 CHF, Weine bitte bis 17 Uhr dekantiert.
```

**Trauben-Vergleich:**
```
Vergleich: Welcher meiner Nebbiolo ist am nächsten am Trinkhöhepunkt?
```

**Keller-Analyse:**
```
Bin ich zu Bordeaux-lastig? Was fehlt mir an Diversität?
```

**Einkauf planen:**
```
Ich gehe morgen zu Gerstl. Was sollte ich kaufen
basierend auf meinen Lücken und aktuellen Aktionen?
```

**Jahresrückblick:**
```
Wie hat sich mein Keller dieses Jahr verändert?
Flaschen gekauft, getrunken, Wert, neue Regionen.
```

**Reise-Briefing:**
```
Ich fahre nächste Woche ins Piemont.
Welche Weine aus der Region habe ich, was ist dort spannend?
```

## 📱 iOS Text Replacements (Schnellzugriff)

Diese Kürzel können unter Einstellungen → Allgemein → Tastatur → Textersetzung eingerichtet werden. Tippe das Kürzel in iMessage und der Text wird automatisch ersetzt:

| Kürzel | Wird ersetzt durch |
|--------|--------------------|
| `;tonight` | Was soll ich heute Abend öffnen? |
| `;pairing` | Was passt zu  |
| `;stats` | Kellerübersicht |
| `;urgent` | Welche Weine muss ich bald trinken? |
| `;wotd` | Wein des Tages |
| `;dinner` | Dinner planen:  |
| `;hilfe` | Was kannst du? Zeig mir Beispiele. |
